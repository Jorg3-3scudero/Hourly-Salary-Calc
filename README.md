# MVC Salary Calculator

## English Description

This project is a desktop application developed in Python using PyQt5. I created it to solve a common problem for freelancers and service providers: calculating monthly income accurately without manually dealing with currency conversions or calculator errors.

The architecture follows the MVC (Model-View-Controller) pattern to keep the code clean, organized, and scalable.

### Objective
To facilitate service providers in knowing their monthly salary based on the number of hours worked and the hourly rate in their country or specific job (keeping in mind this rate varies constantly).

### Key Features
- **MVC Architecture:** Separation of logic, interface, and control.
- **Dynamic Currency:** Select a country (e.g., Colombia, Canada, USA) and the currency symbol updates automatically.
- **Real-time Formatting:** Salaries are displayed with correct punctuation (thousands separators) for better readability.
- **User Friendly:** Clean interface that validates inputs to prevent errors.

### Technologies
- Python 3
- PyQt5 (GUI)
- MVC Pattern

## Descripción en Español

Este proyecto es una aplicación de escritorio desarrollada en Python usando PyQt5. La creé para solucionar un problema común en freelancers y prestadores de servicios: calcular el ingreso mensual con exactitud sin estar peleando con conversiones de moneda o errores de calculadora.

La arquitectura sigue el patrón MVC (Modelo-Vista-Controlador) para mantener el código limpio, organizado y escalable.

### Objetivo
Facilitar a las personas de prestación de servicio conocer su salario mensual basado en la cantidad de horas laboradas y el valor de la hora en tu país/trabajo (revisar bien, ya que esta puede variar).

### Características Principales
- **Arquitectura MVC:** Separación total de la lógica, la interfaz y el control.
- **Moneda Dinámica:** Seleccionas un país (ej. Colombia, Canadá, USA) y el símbolo de moneda se actualiza automáticamente.
- **Formato en Tiempo Real:** Los salarios se muestran con la puntuación correcta (separadores de miles) para facilitar la lectura.
- **Fácil de Usar:** Interfaz limpia que valida las entradas para evitar errores.

### Tecnologías
- Python 3
- PyQt5 (Interfaz Gráfica)
- Patrón MVC

## How to run / Cómo ejecutar

1. Install dependencies / Instalar dependencias:
   pip install -r requirements.txt

2. Run the application / Ejecutar la aplicación:
   python main.py
